Githubio+hugo Theme Blog 구축기.

Dec 21, 2020 · 1 min read

1. hugo 설치

% brew isntall hugo
% hugo version // 휴고 버전 확인

2. Directory 구성

├── archetypes
├── config.toml //hugo theme 설정파일
├── content // post 저장 위치
├── data
├── layouts
├── static //image 저장 위치 인듯...?
└── themes // 테마 저장 폴더

3. hugo로 웹 사이트 생성

% hugo new site <디렉토리명>

4. 테마 다운로드

% cd theme
theme % git clone

5. config.toml 설정

% vi config.toml
theme = "테마명" //적용할 테마명을 넣는다.(필수!!)
# 나머지는 테마에 따라 설정이 달라진다. theme에 저장한 theme의 config.toml을 참조해서 수정해야 함.

6. 글작성

% hugo new <폴더명>/[md파일명].md // content/<폴더명>/[md파일명].md 에 파일 생성 됨.

명령어 실행 후 아래와 같이 파일 내용이 작성된다. categories,tags는 원할 경우 아래와 같이 추가해서 사용해야함.

-–
title: “마크다운 작성법”
date: 2020-12-20T20:19:13+09:00
draft: true
categories: [“category명”]
tags: [“tag명”]
-–

7.1 Disqus 가입 후 Setting 클릭

7.2 Profile, Account 작성

7.3 Home으로 이동 후 Get Started 클릭

7.4 I want to install Disqus on my site 클릭

7.5 Create a new site 작성

7.6 화면을 조금 내리서 Basic에 Subscribe Now 클릭(돈많으면…돈내고쓰세요…ㅠㅠ)

7.7 화면을 조금 내려서 아래 버튼 클릭(hugo는 지원목록에 없다. 추가될수도..?)

7.8 configure 클릭

7.9 사이트 정보 기입

7.10 완료!

7.11 config.toml 수정

% vi config.toml
disqusShortname = "<Disqus shortname>"
